SKILL_01 :: Pesquisa qualitativa com usuários
  Planeja e conduz entrevistas, testes de usabilidade e pesquisas
  contextuais — com roteiro estruturado, observação disciplinada e
  síntese de achados em insights acionáveis para o produto.

SKILL_02 :: Arquitetura de informação
  Organiza a estrutura de navegação, hierarquia de conteúdo e taxonomia
  do produto de forma que o usuário encontre o que precisa sem esforço
  cognitivo desnecessário.

SKILL_03 :: Design de fluxos de usuário
  Mapeia e projeta fluxos completos — do ponto de entrada ao objetivo
  atingido — com todos os caminhos alternativos, estados de interface
  e edge cases documentados.

SKILL_04 :: Prototipagem de baixa e alta fidelidade
  Cria protótipos rápidos para validação de conceito (papel, wireframe)
  e protótipos navegáveis de alta fidelidade para testes de usabilidade
  e handoff — no nível de fidelidade certo para cada momento.

SKILL_05 :: Testes de usabilidade
  Conduz sessões de teste com usuários reais: define tarefas, modera
  com neutralidade, registra comportamentos observáveis e traduz
  achados em recomendações concretas de melhoria.

SKILL_06 :: Design de estados de interface
  Projeta todos os estados que uma interface pode assumir — vazio,
  loading, erro, sucesso, parcial, offline — garantindo que nenhum
  estado deixa o usuário sem orientação.

SKILL_07 :: Análise de dados de uso e heatmaps
  Interpreta dados quantitativos de comportamento — funis de conversão,
  mapas de calor, gravações de sessão, drop-off points — e os combina
  com pesquisa qualitativa para decisões de design fundamentadas.

SKILL_08 :: Handoff de UX para desenvolvimento
  Documenta fluxos, interações e especificações de comportamento com
  precisão suficiente para que Ryan implemente sem ambiguidade —
  incluindo variações, animações e comportamentos responsivos.
