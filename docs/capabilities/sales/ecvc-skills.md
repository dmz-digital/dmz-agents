SKILL_01 :: Revisao de term sheet
  Classifica cada clausula e entrega semaforo geral com relatorio de impacto.

SKILL_02 :: Traducao de termos para linguagem de negocio
  Converte liquidation preference, anti-dilution, pro-rata, drag-along
  para impacto pratico em cada cenario do fundador.

SKILL_03 :: Preparacao para negociacao de clausulas
  Identifica clausulas com espaco para negociar e sugere contrapropostas
  baseadas em pratica VC brasileiro e global.

SKILL_04 :: Analise de SAFE com mapeamento de gatilhos
  Mapeia cap, discount, MFN, maturity e pro-rata. Calcula impacto de cada
  gatilho no cap table pos-conversao.

SKILL_05 :: Revisao de nota conversivel
  Analisa maturity date, juros, discount e cap. Simula diluicao em
  diferentes cenarios de conversao.

SKILL_06 :: Analise de acordo de socios e vesting
  Revisa vesting, cliff, acceleration, ROFR e IP assignment. Identifica
  lacunas que podem criar conflito entre co-fundadores.

SKILL_07 :: Leitura de cap table e projecao de diluicao
  Projeta impacto de nova captacao em cada parte. Calcula diluicao real
  considerando option pool shuffles e SAFEs pre-rodada.

SKILL_08 :: Identificacao de inconsistencias entre verbal e documental
  Compara o acordado nas negociacoes com o que aparece nos documentos.
  Detecta quando rascunho contradiz term sheet ou versoes anteriores.

SKILL_09 :: Redacao de contrapropostas para clausulas problematicas
  Redige a contraproposta com justificativa em pratica de mercado e
  argumento que o fundador pode usar na negociacao com o investidor.

SKILL_10 :: Sinalizacao de quando contratar advogado especializado
  Identifica quando a complexidade supera o escopo de revisao e recomenda
  contratacao de advogado ECVC habilitado para a negociacao final.
