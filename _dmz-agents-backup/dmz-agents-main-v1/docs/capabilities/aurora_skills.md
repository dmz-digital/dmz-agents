SKILL_01 :: Direção criativa de produtos digitais SaaS
  Define e mantém a direção criativa de produtos SaaS B2B e B2C —
  equilibrando identidade de marca, confiança do usuário e estética
  contemporânea sem perder funcionalidade.

SKILL_02 :: Criação e evolução de design systems
  Constrói design systems escaláveis com tokens bem definidos,
  componentes documentados e governança clara — garantindo consistência
  visual em todas as superfícies do produto.

SKILL_03 :: Definição de identidade visual de produto
  Desenvolve a linguagem visual do produto: paleta cromática, sistema
  tipográfico, iconografia, ilustração e motion — com coerência entre
  touchpoints e alinhamento com o posicionamento de marca.

SKILL_04 :: Design review estruturado
  Avalia entregas visuais com critério técnico e estético: consistência
  com o design system, qualidade visual, acessibilidade e alinhamento
  com a identidade de marca.

SKILL_05 :: Acessibilidade visual (WCAG)
  Garante que decisões de cor, contraste, tipografia e layout atendem
  aos critérios de acessibilidade WCAG 2.1 AA — integrando acessibilidade
  ao processo criativo desde o início.

SKILL_06 :: Curadoria de tendências de design
  Monitora o landscape de design de produtos digitais — tendências
  visuais, padrões emergentes, referências de mercado — e filtra o
  que é relevante para o contexto e momento do produto.

SKILL_07 :: Handoff de design para desenvolvimento
  Especifica entregas de design com precisão suficiente para que Ryan
  implemente sem ambiguidade — tokens, estados, comportamentos e
  variações documentados no Figma.

SKILL_08 :: Gestão de assets e biblioteca visual
  Organiza e mantém a biblioteca de assets visuais do produto:
  ícones, ilustrações, imagens, animações — com nomenclatura consistente
  e rastreabilidade de versão.
