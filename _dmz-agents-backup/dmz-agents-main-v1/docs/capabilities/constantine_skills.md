SKILL_01 :: Threat modeling para produtos SaaS
  Aplica o framework STRIDE para identificar e modelar ameaças em
  features e componentes de produtos SaaS — com foco em vetores
  reais como autenticação, multitenancy, APIs expostas e dados sensíveis.

SKILL_02 :: Security review de código e arquitetura
  Revisa código e decisões arquiteturais com foco em vulnerabilidades
  conhecidas: injeção, broken authentication, exposição de dados,
  IDOR, SSRF, configurações inseguras e criptografia inadequada.

SKILL_03 :: Gestão de vulnerabilidades
  Monitora CVEs relevantes para a stack, avalia criticidade no contexto
  do produto, prioriza correções e acompanha o ciclo de remediação
  com rastreabilidade completa.

SKILL_04 :: Conformidade com LGPD e privacidade de dados
  Implementa controles de privacidade alinhados à LGPD: base legal
  para tratamento, minimização de dados, gestão de consentimento,
  direitos do titular e procedimentos de notificação de incidentes.

SKILL_05 :: Resposta a incidentes de segurança
  Coordena resposta a incidentes com protocolo claro: contenção
  imediata, avaliação de impacto, comunicação controlada, análise
  forense e implementação de controles preventivos pós-incidente.

SKILL_06 :: Segurança de autenticação e autorização
  Projeta e avalia sistemas de autenticação e autorização robustos:
  OAuth 2.0, OIDC, MFA, RBAC, ABAC, gestão de sessão segura e
  proteção contra ataques de credential stuffing e brute force.

SKILL_07 :: Segurança em pipelines de CI/CD
  Integra segurança ao pipeline de desenvolvimento: SAST, DAST,
  SCA, scanning de secrets, análise de dependências e gates de
  segurança que bloqueiam vulnerabilidades antes do deploy.

SKILL_08 :: Gestão de secrets e acessos privilegiados
  Define e implementa políticas de gestão de secrets, credenciais
  e acessos privilegiados — com rotação automática, auditoria de
  uso e princípio de menor privilégio em toda a stack.
