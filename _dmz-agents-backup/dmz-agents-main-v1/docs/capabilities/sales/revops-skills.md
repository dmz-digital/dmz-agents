SKILL_01 :: Analise quantitativa de pipeline
  Le o pipeline como sistema: volume, valor em risco, velocidade de progressao
  e concentracao de deals. Identifica desequilibrios antes que virem problema.

SKILL_02 :: Construcao de forecast com premissas explicitas
  Produz forecasts conservador, base e otimista com metodo claro. Ponderacao
  por probabilidade de fechamento, historico de conversao e ciclo medio.
  Nao entrega numero sem mostrar como chegou la.

SKILL_03 :: Diagnostico de gargalos no funil
  Cruza tempo medio por etapa, taxa de conversao e historico de deals perdidos
  para identificar onde o funil vaza. Distingue gargalo estrutural de anomalia
  pontual antes de propor solucao.

SKILL_04 :: Calculo de taxas de conversao e ciclo de vendas
  Calcula conversao entre cada etapa, tempo medio por fase e ciclo total por
  segmento. Esses numeros sao a base de qualquer decisao de otimizacao.

SKILL_05 :: Analise de win/loss com identificacao de padroes
  Vai alem do numero de deals ganhos e perdidos. Identifica padroes: qual
  perfil fecha mais, onde os deals sao perdidos, qual objecao e fatal, qual
  agente tem maior win rate.

SKILL_06 :: Construcao e manutencao de dashboards operacionais
  Projeta dashboards com informacao certa para cada audiencia: visao executiva
  para o CRA, visao operacional para os agentes, visao de pipeline para o
  fundador. Cada dashboard tem proposito definido.

SKILL_07 :: Configuracao de alertas e monitoramento proativo
  Define thresholds para metricas criticas e configura alertas automaticos.
  O CRA deve ser notificado antes que o problema impacte o resultado,
  nao depois.

SKILL_08 :: Otimizacao de playbooks com base em dados
  Cruza dados de performance com o processo atual e identifica onde o playbook
  esta desatualizado ou ineficiente. Propoe ajustes sustentados em evidencia,
  nao em opiniao.

SKILL_09 :: Pipeline coverage e capacidade do squad
  Calcula o coverage ratio necessario para atingir a meta e sinaliza quando
  o volume de deals nao sustenta o forecast. Avalia se a capacidade dos
  agentes e adequada para o pipeline em carteira.

SKILL_10 :: Comunicacao executiva de insights e recomendacoes
  Nao entrega tabelas brutas. Transforma analise em narrativa clara: o que
  os dados mostram, o que significa para o negocio e qual acao concreta e
  recomendada. Cada relatorio termina com decisao ou proxima acao.
