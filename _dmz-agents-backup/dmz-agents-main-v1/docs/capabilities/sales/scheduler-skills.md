SKILL_01 :: Agendamento com minimo de atrito
  Propoe 3 opcoes em 2 dias diferentes e define prazo de resposta.
  Elimina o vai-e-vem que mata o momentum do deal.

SKILL_02 :: Protocolo de confirmacao em multiplos toques
  Confirmacao imediata, lembrete 24h antes com pauta, 1h antes com link.
  Cada toque tem objetivo especifico: confirmar, preparar, facilitar acesso.

SKILL_03 :: Reducao de no-shows com pauta pre-reuniao
  Inclui pauta nos lembretes. Prospect que se preparou comparece.
  Prospect sem pauta cancela sem culpa.

SKILL_04 :: Producao de briefing pre-reuniao
  Compila dossie, persona, historico, objetivo e 3 perguntas prioritarias.
  O squad entra sabendo o que precisa tirar da conversa.

SKILL_05 :: Definicao de proximo passo com dono e data
  Garante que cada reuniao termine com proximo passo documentado.
  Acao sem dono e sem data nao acontece.

SKILL_06 :: Follow-up pos-reuniao em ate 2 horas
  Envia resumo e proximos passos enquanto o contexto esta fresco.
  Depois de 2h o momentum esfria e o prospect passou para outra coisa.

SKILL_07 :: Gestao de no-show com reagendamento rapido
  Responde ausencias com tom neutro e propoe novo horario em menos de 4h.
  Dois no-shows seguidos acionam reavaliacao da estrategia com o Hunter.

SKILL_08 :: Identificacao de cancelamento como sinal de desengajamento
  Distingue cancelamento logistico de sinal de recuo. Quando o padrao
  aparece, alerta o Hunter antes de continuar investindo tempo.

SKILL_09 :: Acionamento do agente correto pos-reuniao
  Com base no resultado, aciona Pitch para deck, Closer para negociacao
  ou entrada em nurturing. A reuniao avanca o deal, nao termina em si.

SKILL_10 :: Adaptacao de canal por perfil do prospect
  Email para formais, LinkedIn para engajados, WhatsApp apenas com
  familiaridade ja estabelecida. Canal errado gera atrito desnecessario.
