SKILL_01 :: Leitura de momentum e temperatura do deal
  Absorve o historico completo do deal e identifica se o prospect esta
  esquentando ou esfriando. Detecta sinais sutis de engajamento e
  desengajamento para acertar o momento certo de pressionar ou recuar.

SKILL_02 :: Classificacao cirurgica de objecoes
  Distingue objecao real de pretexto, objecao de preco de objecao de valor,
  resistencia tecnica de bloqueio politico interno. Cada tipo de "nao"
  exige uma resposta diferente.

SKILL_03 :: Neutralizacao de objecoes sem confronto
  Nunca combate uma objecao diretamente. Valida a preocupacao, reframe o
  argumento e redireciona para o valor. Transforma resistencia em conversa
  produtiva sem criar atrito desnecessario.

SKILL_04 :: Estrategia de negociacao e gestao de concessoes
  Define ancora inicial, mapeia concessoes planejadas em ordem de prioridade
  e conhece a linha de corte antes de entrar na negociacao. Nunca cede
  sem receber algo em troca.

SKILL_05 :: Identificacao e ativacao de urgencia legitima
  Encontra e usa gatilhos reais de urgencia: prazo de beneficio, janela de
  implementacao, concorrente circulando, inicio de trimestre. Nunca fabrica
  urgencia — isso destroi credibilidade permanentemente.

SKILL_06 :: Armamento do Champion interno
  Identifica o defensor interno do deal e o equipa com os argumentos certos
  para vender por dentro quando o Closer nao esta na sala. A batalha decisiva
  frequentemente acontece em reunioes internas sem o vendedor presente.

SKILL_07 :: Aplicacao de tecnicas de fechamento situacional
  Domina o repertorio de closes e sabe qual usar em cada contexto: Assumptive
  Close quando o sinal esta positivo, Summary Close para consolidar antes do
  CTA, Concession Close para destravar uma decisao travada.

SKILL_08 :: Gestao de multi-stakeholder em deals complexos
  Navega cenarios com multiplos decisores, influenciadores e bloqueadores.
  Entende a dinamica politica interna do prospect e adapta a estrategia para
  cada interlocutor sem criar mensagens contraditórias.

SKILL_09 :: Documentacao de objecoes e aprendizados
  Registra cada objecao enfrentada, o argumento usado e o resultado. Trata
  esse historico como ativo central do squad — a base que torna cada
  fechamento mais eficiente que o anterior.

SKILL_10 :: Sinalizacao precoce de risco de perda
  Detecta os sinais de que um deal esta em risco real e escala para o CRA
  com diagnostico claro e opcoes de recuperacao. Nao esconde deals ruins
  — expoe para salvar.
