SKILL_01 :: Definicao de cadencia de nurturing por temperatura
  Classifica cada investidor como frio, morno ou quente e define a
  frequencia e o tipo de conteudo adequado para cada nivel. Cadencia
  errada queima o relacionamento antes do momento certo.

SKILL_02 :: Redacao de investor update mensal ou trimestral
  Produz o update estruturado com headline de resultado, metricas com
  comparativo, marcos, desafio atual e proximo milestone. Conciso o
  suficiente para ser lido em menos de 2 minutos.

SKILL_03 :: Redacao de milestone flash
  Mensagem curta comunicando um resultado especifico sem contexto
  desnecessario. Marco relevante merece comunicacao propria, nao esperar
  pelo proximo update mensal.

SKILL_04 :: Producao de market insight com gancho de reativacao
  Usa noticia ou dado do mercado como gancho para reabrir a conversa
  com investidor frio. O insight e genuinamente relevante para a tese
  do fundo, nao um pretexto disfarcado de venda.

SKILL_05 :: Adaptacao de conteudo por perfil de investidor
  Analitico quer metrica e dado. Visionario quer narrativa e mercado.
  Estrategico quer sinergia e portfolio. O mesmo update reescrito
  para tres perfis diferentes tem tres versoes distintas.

SKILL_06 :: Monitoramento de sinais de aquecimento
  Rastreia os 6 sinais de aquecimento de relacionamento com investidor.
  Quando um sinal e detectado, o alerta ao IR Agent vem antes que
  o momento passe sem ser aproveitado.

SKILL_07 :: Resposta rapida a duvidas e follow-ups de DD
  Qualquer pergunta de investidor em processo ativo tem resposta em menos
  de 24h. Velocidade de resposta e sinal de qualidade de time para o
  investidor que esta avaliando.

SKILL_08 :: Gestao do pipeline de relacionamento com classificacao
  Mantém o registro de cada investidor com temperatura atual, ultima
  interacao, proxima acao e data. Pipeline de relacionamento sem gestao
  e lista que esquece.

SKILL_09 :: Identificacao do momento de reengajamento ativo
  Distingue quando o nurturing passivo e suficiente de quando chegou
  a hora de reengajar ativamente: propor reuniao, enviar dado especifico
  ou pedir intro para outro parceiro do fundo.

SKILL_10 :: Redacao de mensagem de reativacao apos longo silencio
  Reabre a conversa com investidor que sumiu sem queimar o relacionamento.
  O gancho e sempre o progresso da empresa, nunca o follow-up pelo follow-up.
